# Introducción

>Distributed databas system (DDBS)

Es la unión de dos enfoques opuestos en el procesamiento de datos, es decir los sistemas de bases de datos y las redes de computadoras.

Los sistemas de bases de datos distribuidas cambian el paradigma centralizado de los antiguos sistemas, permitiendo que los datos se almacenen y gestionen en múltiples ubicaciones de forma coordinada. Este enfoque busca mantener la independencia lógica y física de los datos, de modo que las aplicaciones puedan acceder a ellos sin preocuparse por su ubicación o forma de almacenamiento.

La integración de datos es un termino que se puede alcanzar sin ayuda de la centralización y eso es lo que busca lograr la bases de datos distribuidas
