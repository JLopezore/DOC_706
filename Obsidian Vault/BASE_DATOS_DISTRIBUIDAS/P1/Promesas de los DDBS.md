## Ventajas
- Transparencia
	- Fragmentación
	- Replicación
	- Independencia de datos
	- Lenguaje
	- Red
- Fiabilidad
- Rendimiento

## Desventajas
- Factores que complican los DDBS
	- Replicación de datos
	- Complejidad
	- Costo
	- Distribución del control
	- Seguridad
- Problemas de Área
	- Diseño
	- Procesamiento de consultas
	- Gestión de Directorios
	- Control de concurrencia
	- Gestión de bloqueos (deadlocks)
	- Fiabilidad de los **DDBMS**
	- Soporte del Sistema Operativo
	- Bases de datos Heterogéneas
